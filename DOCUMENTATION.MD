## Aperçu

**ETNAir** est une plateforme web complète permettant aux utilisateurs de :

- Parcourir des annonces de locations (appartements, maisons, studios…)
- Réserver un bien directement en ligne
- Publier leurs propres annonces
- Créer et supprimer leurs annonces existantes
- Visualiser l'emplacement exact de chaque bien sur une carte interactive

---

## Fonctionnalités principales

### Créer un compte

Pour créer un compte, il suffit de cliquer sur le bouton *Iscription* t de remplir les champs d'informations suivants :
- Prénom
- Nom de famille
- Adresse email
- Mot de passe (Attention : le mot de passe doit contenir au minimum 8 caractères)

---

### 🔍 Consulter les annonces

La page d'accueil affiche l'ensemble des annonces disponibles sous forme de cartes visuelles.

Chaque annonce contient :
-  Photo(s) du logement
-  Localisation (ville)
-  Prix par nuit
-  Description du logement avec les équipements à disposition
-  Note et avis des locataires précédents
-  Disponibilités

Des **filtres de recherche** permettent d'affiner les résultats par :
- Localisation
- Prix max par nuit
- Prix croissant/décroissant
---

### 📅 Réserver un logement

Une fois que vous avez cliquez sur une annonce pour voir les détails, il est possible de **réserver le logement** en quelques étapes :

1. Sélectionner les **dates d'arrivée et de départ**
2. Vérifier le **récapitulatif de la réservation** (prix total, conditions)
3. Confirmer la réservation en appuyant sur le bouton *Rserver*
 ⚠️ Une **connexion à un compte utilisateur** est requise pour finaliser une réservation.

Une fois confirmée, la réservation apparaît dans votre espace personnel sous **"Mes réservations"**.
---

### ➕ Créer une annonce

Les utilisateurs connectés peuvent publier leurs propres biens à louer depuis leur espace personnel. Pour cela, vous devez d'abord avoir créer un compte *Proriétaire* su le site, puis sur votre profil, cliquez sur le bouton *Pubier une annonce*.

*Formulaire de création d'annonce :**

| Champ | Description |
|---|---|
| Titre | Nom accrocheur de l'annonce |
| Description | Présentation détaillée du logement (superficie, équipement, type de logement) |
| Adresse | La ville où est située le logement |
| Prix | Tarif par nuit ou par mois |
| Photos | Upload de plusieurs photos (formats JPG, PNG) |
| Disponibilités | Plages de dates disponibles à la location |

Une fois soumise, l'annonce est **publiée immédiatement** et visible par tous les visiteurs.

---

### 📁 Gestion des annonces

Depuis la section **"Mes annonces"** de votre espace personnel, vous pouvez gérer toutes vos publications :

- **Modifier** une annonce existante (titre, prix, photos, disponibilités…)
- **Supprimer définitivement** une annonce

> ⚠️La suppression d'une annonce est **irréversible**. Une confirmation est demandée avant toute suppression.

Les réservations déjà confirmées sur cette annonce ne sont pas affectées par sa suppression.
---

### 🗺️ Carte interactive

Chaque annonce est géolocalisée grâce à une **carte interactive intégrée** (basée sur [Leaflet.js](https://leafletjs.com/).

**Fonctionnalités de la carte :**

- 📌 Marqueur de localisation** précis pour chaque annonce
- 🔎 Zoom** et déplacement libre sur la carte
- 🗂️*Vue liste + carte** simultanée : les résultats de recherche s'affichent sur la carte en temps réel
- 📍 Géolocalisation de l'utilisateur** pour trouver les logements à proximité
